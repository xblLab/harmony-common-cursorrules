# 鸿蒙原生 app 开发通用 cursorrules

## 使用方法

### cursor 使用方法

先将 `symbolname.cursorrules` 复制到项目中。

将 `.cursorrules.example` 文件放到你项目根目录，重命名为 `.cursorrules` 即可。

如果项目中已经存在 `.cursorrules` 文件，你可以选择：

1. 将 `.cursorrules.example` 文件中的内容添加到 `.cursorrules` 文件中即可。
2. 将 `.cursorrules.example` 保存为 `harmony-common-cursorrules.cursorrules` 文件，然后在 `.cursorrules` 文件中添加以下说明：

    ```txt
    在开发鸿蒙原生 app 时，请参考 harmony-common-cursorrules.cursorrules 文件中的内容。
    ```

先将 `symbolname.cursorrules` 复制到项目中。

将 `CLAUDE.md.example` 文件放到你项目根目录，重命名为 `CLAUDE.md` 即可。

如果项目中已经存在 `CLAUDE.md` 文件，你可以选择：

1. 将 `CLAUDE.md.example` 文件中的内容添加到 `CLAUDE.md` 文件中即可。
2. 将 `CLAUDE.md.example` 保存为 `harmony-common-cursorrules.claude.md` 文件，然后在 `.cursorrules` 文件中添加以下说明：

    ```txt
    在开发鸿蒙原生 app 时，请参考 harmony-common-cursorrules.claude.md 文件中的内容。
    ```
